### Description
This field will show a list of ISO 4217 currencies (the value is the ISO 3 letters code)

### Sample code
```
<field 
   name="currency" 
   type="rcurrency"
   label="COM_XXX_FIELD_CURRENCY"
   description="COM_XXX_FIELD_CURRENCY_DESC"
   class="chosen"
>
   <option value="">JOPTION_SELECT_CURRENCY</option>
</field>
```
