### Description
This field will show a list of ISO 3166 countries (the value is the ISO 2 letters code)

### Sample code
```
<field 
   name="country" 
   type="rcountry"
   label="COM_XXX_FIELD_COUNTRY
   description="COM_XXX_FIELD_COUNTRY_DESC"
   class="chosen"
>
   <option value="">JOPTION_SELECT_COUNTRY</option>
</field>
```
